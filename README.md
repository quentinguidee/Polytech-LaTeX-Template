## Requirements

To use this template, you have to install the free [Jaapokki font](https://mikkonuuttila.com/jaapokki/) made by [Mikko Nuuttila](https://www.behance.net/MikkoNuuttila) and use Xelatex.

## Colors

To change the main color

```
\maincolor{color_xxx}
```
